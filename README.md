Prerequisits

```
sudo apt install libsdl2-dev libsdl2-mixer-dev libsdl2-net-dev
```

Then,

```
./autogen.sh
make -j4
```

Finally from `chocolate-doom` folder

```
src/chocolate-doom -iwad ../misc/OriginalWADs/DOOM2.WAD
```

Enjoy!
